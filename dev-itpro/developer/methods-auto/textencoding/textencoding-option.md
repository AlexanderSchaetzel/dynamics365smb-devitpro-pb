# TextEncoding Option Type
> **Version**: _Available from runtime version 1.0._

Represents a file encoding.

|  Member  |  Description  |
|----------------|---------------|
|MSDos|MSDos encoding.|
|UTF8|UTF8 encoding.|
|UTF16|UTF16 encoding.|
|Windows|Windows encoding.|
