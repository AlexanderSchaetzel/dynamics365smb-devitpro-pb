# Version Property
> **Version**: _Available from runtime version 1.0._

Specifies the version of the .NET assembly.

## Applies to
-   Dot Net Assembly


[//]: # (IMPORTANT: END>DO_NOT_EDIT)


## Syntax

```AL

```
