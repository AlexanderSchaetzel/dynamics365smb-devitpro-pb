# Multiplicity Property
> **Version**: _Available from runtime version 6.1._

Specify the multiplicity of the part.

## Applies to
-   Page Part

## Property Value

|Value|Description|
|-----------|---------------------------------------|
|**ZeroOrOne**|Zero or one entity|
|**Many**|Any number of entities.|
