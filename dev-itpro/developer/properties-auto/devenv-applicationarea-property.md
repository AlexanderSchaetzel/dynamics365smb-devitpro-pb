# ApplicationArea Property
Sets the application areas that apply to the control. Standard values are All, Basic, Suite, and Advanced.

## Applies to
-   Page Label
-   Page Field
-   Page Part
-   Page System Part
-   Page Chart Part
-   Page Action
-   Page User Control
-   Page
-   Report
