# IsHeader Property
Specifies if the page action separator is a header.

## Applies to
-   Page Action Separator

[//]: # (IMPORTANT: END>DO_NOT_EDIT)
## Property Type

**True** if the page action separator is a header; otherwise, **false**.


## Syntax

```AL
IsHeader = true;
```
