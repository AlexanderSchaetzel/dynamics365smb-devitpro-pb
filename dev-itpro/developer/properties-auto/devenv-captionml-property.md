# CaptionML Property
Sets the text string that displays with the object, control, or other element in user interface for the current language.

## Applies to
-   Table
-   Field
-   Page Field
-   Field Group
-   Page
-   Request Page
-   Page Label
-   Page Group
-   Page Part
-   Page System Part
-   Page Action
-   Page Action Group
-   Page Action Separator
-   Xml Port
-   Report
-   Query
-   Query Column
-   Query Filter
-   Report Column
-   Enum Value
-   Page View
-   Profile
-   Enum Type
