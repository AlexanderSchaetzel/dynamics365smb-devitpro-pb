# MultiLine Property
Sets the value that indicates whether a field can display multiple lines of text.

## Applies to
-   Page Label
-   Page Field

[//]: # (IMPORTANT: END>DO_NOT_EDIT)
## Property Value  

**True** if you want to display multiple lines; otherwise, **false**. The default is **false**.  

## Syntax

```AL
MultiLine = true;
```
