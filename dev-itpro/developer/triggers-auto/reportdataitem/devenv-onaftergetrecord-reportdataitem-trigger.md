# OnAfterGetRecord (Report Data Item) Trigger
> **Version**: _Available or changed with runtime version 1.0._

Runs each time a record is retrieved from the data item.



## Syntax
```
trigger OnAfterGetRecord()
begin
    ...
end;
```
