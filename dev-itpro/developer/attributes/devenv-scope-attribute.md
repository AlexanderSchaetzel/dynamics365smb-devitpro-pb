# Scope Attribute
> **Version**: _Available or changed with runtime version 1.0._

Specifies the scope of a method.

## Syntax
```
[Scope(Kind: Text)]
```

### Parameters

*Kind*  
&emsp;Type: [Text](../methods-auto/text/text-data-type.md)  
The scope of the method.  


## Applies To

- Method
